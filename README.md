#  Weather Tracker App

##  Project Overview
The Weather Tracker App is a web-based application that allows users to check real-time weather information of any city.  
It fetches live weather data from an external API and displays details like temperature, humidity, and weather conditions.

##  Objective
The objective of this project is to learn how to:
- Work with APIs (fetch real-time data)
- Handle user input dynamically
- Update UI based on fetched data

##  Features
- Search weather by city name  
- Real-time temperature display  
- Shows humidity and weather conditions  
- Dynamic UI updates based on data  
- Error handling for invalid city names  

##  Technologies Used
- HTML 
- CSS  
- JavaScript  
- Weather API (e.g., OpenWeatherMap)

##  Project Structure
- index.html → Main webpage  
- style.css → Styling  
- script.js → API handling & logic  

##  API Used
This project uses a weather API to fetch real-time data.  
Example: OpenWeatherMap API  
- Endpoint: `https://api.openweathermap.org/data/2.5/weather`  
- Requires an API key  

##  How to Run
1. Download or clone the repository  
2. Open the project folder  
3. Add your API key in `script.js`  
4. Open `index.html` in browser  
   OR use Live Server  

##  Note
Make sure to replace the API key with your own for proper functioning.

##  Learning Outcome
- Learned how to integrate APIs  
- Improved JavaScript skills (fetch, async/await)  
- Understood DOM manipulation
